Javascript code:

            
                var today = new Date();
                var hourNow = today.getHours();
                var greeting;
                
                if (hourNow > 18) {
                    greeting = 'Good evening!';
                } else if (hourNow > 12) {
                    greeting = 'Good afternoon!';
                } else if (hourNow > 0) {
                    greeting = 'Good morning!';
                } else {
                    greeting = 'Welcome!';
                }
                
                document.write('<h3>' + greeting + '</h3>');

Output

C01 explanation

In this script we create variables with date and todays hour. The script checks what hour it currently is and gives output corresponding to the correct time of the day.

If it's before 18 it will say 'good evening', before 12 it will output 'good afternoon', if it's this our less than 0 it will output 'Good morning'.

If it's anything else than these hours it will output 'welcome'.

Everything is outputted as a h3 element with variable greeting
